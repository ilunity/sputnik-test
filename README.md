# Тестовое задание на стажировку в "Спутник"

## Запуск 

- `npm install` - установка зависимостей
- `npm run start` - сборка проекта
- `npm run test:run` - запуск тестов

## Стек
  - React
  - TypeScript
  - `Архитектура` 
    - FSD
  - `Стили`
    - AntDesign
    - Styled Components
  - `Работа с данными`
    - Zustand
    - React Query
  - `Тестирование`
    - Vitest
   
## Тестирование

Тесты описаны для следующих фич:
- Добавление задачи
- Изменение статуса задачи
- Удаление задачи
