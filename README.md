# Тестовое задание на стажировку в "Спутник"

## Запуск 

- `npm install` - установка зависимостей
- `npm run start` - сборка проекта
- `npm run preview` - запуск превью проекта на `http://localhost:5173/`

## Стек
  - React
  - TypeScript
  - `Архитектура` 
    - FSD
  - `Стили`
    - AntDesign
    - Styled Components
  - `Работа с данными`
    - Zustand
    - React Query
  - `Тестирование`
    - Vitest
